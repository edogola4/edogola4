## About

Software Engineer based in Nairobi, Kenya, with 2+ years shipping production systems across fintech, healthcare, civic tech, and enterprise SaaS. 

Most recently at **Cognativ Technology Limited** (Nov 2025–Jan 2026), I shipped RESTful APIs for SmartSaaS™—an enterprise cloud platform built on ASP.NET Core/C#. Cut API response times by **35%** through SQL Server query restructuring and EF Core migrations, and drove test coverage to **80%** with xUnit.

Currently building **SmartSchedule Healthcare** full-time—an AI-powered healthcare scheduling SaaS targeting 40% no-show reduction and 50% admin efficiency gains—and actively developing **ChamaBot**. Open to on-site, hybrid, and remote-first roles globally.

## 🌟 Key Projects

### 🏥 SmartSchedule Healthcare
**[github.com/edogola4/smartschedule-healthcare](https://github.com/edogola4/smartschedule-healthcare)**

AI-powered healthcare scheduling SaaS on .NET 10 + Azure. **Targeting** 40% no-show reduction and 50% admin efficiency gains (MVP Q2 2026). Microservices, CQRS/Event Sourcing, HIPAA-compliant Terraform IaC.

**Tech:** `.NET 10` `ASP.NET Core` `Azure` `Blazor` `PostgreSQL` `CQRS` `Event Sourcing` `Terraform` `Microservices`

---

### 🛍️ Riggs London Kenya
**[github.com/edogola4/riggs-london-kenya](https://github.com/edogola4/riggs-london-kenya)**

Solo e-commerce monorepo: Next.js 14 + Fastify + PostgreSQL with pgvector. M-Pesa STK Push + Pesapal payments, Claude 3.5 Haiku product chatbot, semantic search via OpenAI embeddings. Deployed on Railway + Vercel at ~$108/month.

**Tech:** `Next.js 14` `Fastify` `PostgreSQL` `pgvector` `M-Pesa API` `Pesapal` `Claude API` `OpenAI Embeddings` `Railway` `Vercel`

---

### 💬 ChamaBot
**[github.com/edogola4/chamabot](https://github.com/edogola4/chamabot)**

WhatsApp-native savings automation for Kenya's 300,000+ registered chamas. M-Pesa contribution tracking via Daraja API, multilingual intent parsing with Claude API, Redis-based idempotent payment handling. **In Development.**

**Tech:** `WhatsApp Business API` `Claude API` `M-Pesa Daraja` `Redis` `Node.js` `Python`

---

### 🇰🇪 Kenya Civic Info Bot
**[brandonogola.dev/projects/civic-info-bot](https://brandonogola.dev/projects/civic-info-bot)**

RAG-powered civic education chatbot grounded in Kenya's Constitution 2010 and IEBC data. Next.js 14 + Claude API + pgvector on Neon.

**Tech:** `Next.js 14` `Claude API` `RAG Pipelines` `pgvector` `Neon` `PostgreSQL`
